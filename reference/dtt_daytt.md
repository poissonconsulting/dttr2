# Dayte Time

Dayte Time

## Usage

``` r
dtt_daytt(x, start = 1L)
```

## Arguments

- x:

  A Date or POSIXct vector.

- start:

  An integer vector of the starting month or a Date vector of the
  starting date.

## Value

A Date or POSIXct vector with the year for February 29th as 1972.
