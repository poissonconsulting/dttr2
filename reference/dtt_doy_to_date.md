# Day of the Year to Date

Day of the Year to Date

## Usage

``` r
dtt_doy_to_date(x, year = 1972L)
```

## Arguments

- x:

  An integer vector of the Day of the Year.

- year:

  An integer scalar or vector of the year.

## Value

A Date vector.

## Examples

``` r
dtt_doy_to_date(3)
#> [1] "1972-01-03"
```
