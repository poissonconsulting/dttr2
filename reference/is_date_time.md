# Is Date/Time

Tests whether an object is a Date, POSIXct, or hms vector.

## Usage

``` r
is.POSIXct(x)

is_date_time(x)

is.Date(x)

is_date(x)

is.hms(x)

is_time(x)
```

## Arguments

- x:

  An object

## Value

A flag indicating whether x inherits from Date, POSIXct or hms.
